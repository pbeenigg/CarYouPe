一、配色方案
主色调：
背景色：深空黑
强调色：爱马仕橙
辅助色：银色，灰色，
 - 灰色系：#2A2A2A、#3A3A3A（卡片、分区背景）
 - 高光灰：#9CA3AF（次要文字）
 - 纯白：#FFFFFF（主要文字）
点缀色：金属银（#C0C0C0）用于产品特性展示

注意：全局的字体颜色搭配和全局的背景色搭配， 字体要适中

二、网站结构与导航菜单

┌─────────────────────────────────────┐
│  Logo    [首页] [产品] [效果] [关于我们]  │
└─────────────────────────────────────┘
建议导航菜单：

首页 (Home) - 品牌展示与核心卖点 （banner要有设计动效）
产品系列 (Products) - 产品列表
效果展示 (Showcase) - 效果展示
关于我们 (Contact Us) - 合作咨询表单

四、技术栈建议
基于您的需求（静态数据 + 邮件提交），我建议：

前端框架：
React + Next.js（适合静态站点生成 SSG）

样式方案：
Tailwind CSS（快速实现暗色主题）
Framer Motion（页面动画效果）
数据管理：

全局配置文件：config/site-data.json 或 .ts
```json

{
  "hero": {
    "title": "CarYou Pe",
    "subtitle": "一键匹配爱车的每个细节"
  },
  "products": [...],
  "features": [...]
}
```

邮件发送：
EmailJS（纯前端发送邮件，无需后端）
或使用 Next.js API Routes + Nodemailer

五、响应式断点
- Mobile: < 640px (1列布局)
- Tablet: 640px - 1024px (2列布局)
- Desktop: > 1024px (3-4列布局)

六、页面特效建议
增强用户体验的动效：
视差滚动（Parallax）- Hero区域
淡入动画（Fade In）- 模块进入视口时
Hover悬停效果 - 产品卡片、按钮
平滑滚动 - 锚点跳转
加载动画 - 页面切换时（跑车LOGO旋转）
