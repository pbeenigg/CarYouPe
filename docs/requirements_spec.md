# 汽车用品之家 (CarYouPe) 需求规格说明书

## 1. 项目概述

本项目旨在构建一个覆盖微信、小红书、抖音小程序的多端汽车配件售卖平台。平台连接用户（会员）、经销商（门店）和区域代理（合伙人），通过分销机制和 AI 渲染技术提供优质的汽车用品购买体验。

## 2. 核心模块与功能

### 2.1 小程序端

#### 2.1.1 首页

- **基础组件**：轮播图、专区板块、搜索栏。
- **活动运营**：积分中心、福利活动、活动板块。
- **商品展示**：新品展区、热门车型板块、专题板块、商品推荐。

#### 2.1.2 分类

- **商品分类**：按品类（如脚垫、座套、电子设备等）分类。
- **门店分类**：按服务类型或区域分类。
- **车型分类**：按品牌、车系、年代款分类（核心入口）。

#### 2.1.3 购物车

- 支持普通商品、门店物料、赠品一同结算。
- 支持修改数量、删除商品、选择优惠券。

#### 2.1.4 个人中心 (我的)

- **订单管理**：全部、待付款、待发货、待收货、已完成、售后。
- **资产管理**：
  - **账户钱包**：
    - **积分**：1 元 = 1000 积分（用于提现或抵扣，提现需审核/企业打款）。
    - **余额**：人民币余额。
    - **提现规则**：每周三 9:00-23:00 开放提现，走微信企业打款通道。
  - **优惠券**：满减券、折扣券。
- **常用功能**：个人资料、常用地址、公众号链接、等级展示、注册地绑定。
- **角色申请**：申请成为门店经销商、申请成为合伙人。

#### 2.1.5 门店模块

- 门店物料展示。
- 门店板块管理（针对门店角色）。
- 服务内容展示、联系方式、地址及定位导航。

### 2.2 特色功能

- **AI 汽车脚垫预览**：
  - 根据用户选择的车型，利用 AI 渲染技术生成脚垫安装效果图。
  - 支持不同颜色、材质的实时效果切换。
- **分销系统**：
  - **链路**：会员（用户） -> 门店（经销商） -> 合伙人（区域代理） -> 平台。
  - **返佣**：获返佣金可自动转换为积分，积分可提现。
- **分享推广**：
  - 按角色（会员/门店/合伙人）生成定制化推广页/海报。
- **内容社区**：
  - 图文/视频推广资讯。
  - 互动：评论、收藏、点赞。
  - 安全：图文视频防涉黄、敏感词过滤。

## 3. 业务角色与权益体系

### 3.1 角色定义

1.  **会员 (用户)**：普通消费者。
2.  **门店 (经销商)**：实体店或分销商，拥有进货/物料支持。
3.  **合伙人 (区域代理)**：负责区域市场管理。

### 3.2 角色权益详解

#### 3.2.1 合伙人 (区域代理)

- **分级**：省代、市代、区代。
- **分润比例** (净利润分成)：
  - 省代：80%
  - 市代：60%
  - 区代：40%
- **功能权限**：会员/门店管理、关系链管理、推广设置、代理信息管理（企业/个人）。

#### 3.2.2 会员

- **等级**：SVIP, VIP。
- **购物优惠**：
  - SVIP：7.8 折。
  - VIP：8.8 折。
- **推广权益** (SVIP 专属)：
  - 首单：享净利润 30% 分成。
  - 次单：享净利润 50% 分成。
  - 三单及以上：享净利润 80% 分成。
- **会籍管理**：权益查看、续费、有效期管理。

#### 3.2.3 门店

- **权益**：享受供货价（批发价）、增加平台曝光量、线下引流。
- **功能**：物料管理、商品管理、优惠设置、会员管理。

## 4. 支撑系统

### 4.1 登录与认证

- 方式：微信授权登录、手机号验证登录。
- 绑定：手机号绑定、实名认证（提现必需）。

### 4.2 支付能力

- 微信支付。
- 积分支付（混合支付待确认）。
- 银联支付（备选）。

### 4.3 数据分析

- 维度：销量统计、财务报表、访问量分析、分享推广效果分析。

### 4.4 权限管理 (RBAC)

- 基于角色的动态权限控制，不同角色看到不同的功能菜单。

### 4.5 国际化 (i18n)

- **支持语言**：中文 (zh-CN)、英文 (en-US)。
- **覆盖范围**：
  - 页面静态文字 (菜单、标题、标签)。
  - 交互组件文字 (按钮、Placeholder)。
  - 系统反馈信息 (Toast 提示、Modal 对话框)。
  - 异常与错误提示 (API 错误码映射)。
  - 动态通知消息 (站内信、推送)。
- **切换方式**：用户可在设置中手动切换语言，默认跟随系统语言。
